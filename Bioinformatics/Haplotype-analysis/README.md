## 基因单倍型分析

### 简介

单倍型是指同一染色体上不同变异位点的线性组合形式。基因单倍型则特指基因区域内（包括启动子、外显子、内含子、终止子）的不同变异位点组合。本分析使用 geneHapR 包进行基因单倍型的鉴定和分析。

### 数据要求

#### 1. 基因型数据（必需）

基因型数据可以通过以下方式获取：
- 公共数据库下载（推荐，零成本）
- 一代测序（适合基因较短、样本量少的情况）
- 基因芯片或高通量测序（适合大规模分析）

支持的数据格式：
- 一代测序：fasta 格式序列文件
- 二代测序：VCF、Hapmap、plink（map&ped）格式
- 自定义表格：包含 Chr、POS、REF、Alt、INFO 等列的变异位点信息

#### 2. 基因注释信息（可选）

用于查看变异位点在基因上的分布位置。可以从以下来源获取：
- 公共数据库：如 phytozome、maizeGDB 等
- 自定义注释：当数据库注释不准确时

支持的格式：
- GFF/GFF3：标准注释格式
- BED6：自定义格式，包含：
  - 染色体名称
  - 起始位置
  - 终止位置
  - 片段名称（基因ID + 空格 + CDS/UTR）
  - 空列
  - 方向（+/-）

#### 3. 样本信息（可选）

用于进行多种分析：
- 表型数据：评估优势单倍型（如高产、耐逆等）
- 地理坐标：分析单倍型地理分布
- 分类信息：用于单倍型网络分析分组

数据格式要求：
- 文本文件：每行一个样本，列为不同类型的信息
- R数据框：行名为样本名，列名为信息类型（type、subgroup、location等）
